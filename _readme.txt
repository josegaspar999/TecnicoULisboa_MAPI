
SVN Repository for the Industrial Automation Course

José Gaspar

---------------------------------------------------

This repository is designed to work mostly with Matlab

-- Install for a first use in Matlab
Navigate Matlab to the subfolder "_init_all\." and run
>> login_z_install
>> login_mapi

-- Updates
In Matlab run
>> login_z_svnup

-- Usage required every time Matlab is restarted
In Matlab run
>> login_mapi

-- Call the Petri net editor
In Matlab run
>> pn_editor


Extra notes:
------------

The repository is expected to be checked out to "c:\users2\api\".
Consider using the freeware TortoiseSVN to do this checkout.
The URL for the repository is:
svn://svn.isr.tecnico.ulisboa.pt/dccal/projects/lsdc4_mapi

Each group of students is expected to work in a folder
	"c:\users2\apiYY\D9\."
where YY indicates current year and D9 can be A1, A2, ..., B1, ..., C1, ...
